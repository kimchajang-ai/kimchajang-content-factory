# Architecture Overview

## 시스템 개요
김차장 채널의 콘텐츠 생산을 Skills 기반 에이전트 체인으로 자동화한다.

## 핵심 컴포넌트
- Topic Research Agent
- Fact Check Agent
- Scriptwriting Agent
- Audio Package Agent
- Visual Storyboard Agent
- Summary Visual Agent
- Thumbnail Package Agent
- Upload Package Agent
- QA Agent

- Gemini 3 Flash Preview 고정
- 김차장 reference 고정 (하늘색 셔츠, 회색 슬랙스, 사각 안경)
- 화이트 배경 고정
- 한국어 현지화
- 4분 기본
