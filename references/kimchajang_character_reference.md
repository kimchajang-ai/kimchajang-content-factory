# 김차장 캐릭터 레퍼런스 (Kim Chajang Character Reference)

## 👤 외형 (Appearance)
대표님이 주신 캐릭터 시트를 바탕으로 한 공식 외형 설정입니다.

- **인물**: 중년 한국 남성 (40대 후반~50대 초반 인상)
- **머리**: 검은색 짧은 머리, 이마가 약간 넓고 M자형으로 살짝 벗겨짐.
- **안경**: 검은색 굵은 뿔테 안경 (직사각형 형태).
- **의상**:
  - **상의**: 하늘색(Light Blue) 긴팔 셔츠. 소매를 팔꿈치까지 두 번 정도 걷어붙인 형태.
  - **하의**: 짙은 회색(Dark Gray) 슬랙스.
  - **벨트**: 검은색 단순한 디자인의 가죽 벨트.
  - **신발**: 갈색(Brown) 구두 또는 캐주얼 가죽 신발.
- **특징**: 인자하면서도 날카로운 눈매. 입가에 옅은 미소를 띠고 있으나 팩폭 시에는 단호한 표정.

## 🎨 아트 스타일 (Art Style)
- **펜선**: 굵고 명확한 검은색 외곽선 (Thick Black Outlines).
- **채색**: 평면적이고 깔끔한 디지털 채색 (Flat Color with Simple Shading).
- **구도**: 전신(Full Body) 캐릭터를 기본으로 하되, 상반신 클로즈업 가능.
- **배경**: 순백색(Pure White) 또는 아주 단순한 인포그래픽 배경.

## ⚠️ 주의 사항
- 다른 색상의 셔츠나 바지 착용 금지.
- 안경 누락 금지.
- 배경에 복잡한 풍경 넣지 말 것 (화이트 배경 엄수).
