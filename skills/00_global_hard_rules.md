# Global Hard Rules

## 시스템 필수 규칙
이 규칙은 모든 에이전트가 최우선으로 준수해야 하며, 어떠한 경우에도 무시될 수 없습니다.

1. **TTS 보이스 고정**: 반드시 `Sadaltager`를 사용한다. (변경 절대 불가)
2. **이미지 생성 모델**: `gemini-3-flash-preview`를 기본으로 사용한다.
3. **배경 스타일**: 모든 배경은 `화이트(White)` 또는 `화이트 인포그래픽` 스타일로 고정한다.
4. **언어**: 모든 출력물과 소통은 한국어를 원칙으로 한다.
5. **캐릭터 일관성**: 김차장 캐릭터 레퍼런스를 100% 준수한다.
6. **런타임**: 기본 영상 길이는 4분(240초)을 기준으로 설계한다.
